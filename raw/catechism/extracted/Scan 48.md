# 2. ሰው እግዚአብሔርን የሚናፍቀው ለምንድነው?

## 2. ሰው እግዚአብሔርን የሚናፍቀው ለምንድነው?

**Q:** 2. ሰው እግዚአብሔርን የሚናፍቀው ለምንድነው?

**A:** እግዚአብሔር ራሱ ሰውን በራሱ አምሳል ሰለፈጠረ፤ እርሱን የሚያይበትን ናፍቆት ብዙ ጊዜ ችላ ቢልም እንኳን እግዚአብሔር ሰውን ወደራሱ መሳቡን ከቶ አያቋርጥም፤ ምክንያቱም ያለማቋረጥ የሚፈልገውን የእውነትና የደስታ ሙላት የሚያገኘውና በእርሱም የሚኖርበት በእግዚአብሔር ብቻ ነው። ስለዚህ፤ በተፈጥሮም ሆነ በመንፈሳዊ ጥሪ ሰው ከእግዚአብሔር ጋር ህብረት ሊኖረው የሚችል መንፈሳዊ ፍጡር ነው። ይህም ከእግዚአብሔር ጋር ያለው ቅርብና ወሳኝ ትስስር ለሰው ልጅ መሠረታዊ ክብርን ያጎናጽፈዋል።

[CCC 27-30, 44-45]

## 3. በሰው አእምሮ ብቻ እግዚአብሔርን እንዴት ማወቅ ይቻላል?

**Q:** 3. በሰው አእምሮ ብቻ እግዚአብሔርን እንዴት ማወቅ ይቻላል?

**A:** ከፍጥረት ማለትም ከዓለምና ከሰው ልጅ መፈጠር ተነሥቶ ሰው በአእምሮው ብቻ እግዚአብሔር የዓለም መጀመሪያና መጨረሻ፤ ቸርነቱ የባዛ፤ ወሰን የሌለው እውነትና ውበት መሆኑን በእርግጠኝነት ሊያውቅ ይችላል።

[CCC 31-36, 46-47]

## 4. የአእምሮ ብርሃን ብቻ የእግዚአብሔርን ምስጢር ለማወቅ ያስችላል?

**Q:** 4. የአእምሮ ብርሃን ብቻ የእግዚአብሔርን ምስጢር ለማወቅ ያስችላል?

**A:** ሰው በአእምሮው ብቻ እግዚአብሔርን ለማወቅ ብዙ ችግሮች ይጋጠሙታል። በመሠረቱ፤ ሰው በራሱ ወደ መለኮታዊ ምስጢር መግባት አይችልም። ለዚህም ነው ከዕውቀቱ በላይ የሆኑትን ነገሮች ብቻ ሳይሆን በራሳቸው ከሰው አእምሮ በላይ ያልሆኑትን መንፈሳዊና ግብረገባዊ እውነታዎችንም ለማወቅ የእግዚአብሔር የመገለጥ ብርሃን የሚያስፈልገው። ያን ጊዜ የሰው ዘር አሁን ባለበት ሁኔታ፤ እንዲህ በቀላሉ፤ በእርግጠኝነትና ምንም ዓይነት ስህተት ሳይገባበት ለማወቅ ይችላል።

[CCC 37-38]
